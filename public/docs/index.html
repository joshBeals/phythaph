Introduction

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Base URL

http://phythaph.herokuapp.com

Authenticating requests

This API is not authenticated.

Payment APIs

GET api/transaction/initialize

Example request:
curl --request GET \
    --get "http://phythaph.herokuapp.com/api/transaction/initialize" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"amount\": \"libero\",
    \"type\": \"adipisci\",
    \"scope\": \"totam\",
    \"description\": \"voluptas\"
}"

const url = new URL(
    "http://phythaph.herokuapp.com/api/transaction/initialize"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "amount": "libero",
    "type": "adipisci",
    "scope": "totam",
    "description": "voluptas"
};

fetch(url, {
    method: "GET",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

Example response (200, Ok):


{
   "success": true,
   "message": "Transaction generated",
   }

Example response (400, Bad request):


{
    "success": false,
    "message": "Cannot initialize transaction, please check your input and try again"
